# API Reference

This page displays the complete FluentCart API documentation generated from the OpenAPI specification.

## Overview

The FluentCart API provides comprehensive endpoints for managing your e-commerce operations including files, orders, products, customers, and more.

## Using the API

All endpoints require authentication using WordPress Application Password or Cookie authentication.

## Endpoints

### Files

- [List Files](/operations/list-files) - Retrieve a paginated list of uploaded files

### Orders

- [List Orders](/operations/orders/list-orders) - Retrieve a paginated list of orders
- [Create Order](/operations/orders/create-order) - Create a new order
- [Get Order](/operations/orders/get-order) - Get order details
- [Update Order](/operations/orders/update-order) - Update an order
- [Delete Order](/operations/orders/delete-order) - Delete an order
- [Mark as Paid](/operations/orders/mark-as-paid) - Mark order as paid
- [Refund Order](/operations/orders/refund-order) - Process a refund
- [Update Statuses](/operations/orders/update-statuses) - Update order statuses
